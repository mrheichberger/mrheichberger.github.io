Mathematical Derivations (Planned + Active)

Each block below captures the primary objective and core equation used to derive implementation updates.

LMS

w_{t+1} = w_t + \mu e_t x_t

Perceptron

w_{t+1} = w_t + \eta y_t x_t, if y_t(w_t^T x_t) \le 0

Fisher LDA

w^* = S_w^{-1}(\mu_1 - \mu_2)

Bayesian Classifier

\hat{y} = argmax_k p(x|C_k)P(C_k)

PCA

\max_{||u||=1} u^T \Sigma u

SVM

\min_{w,b,\xi} \frac{1}{2}||w||^2 + C\sum_i \xi_i

K-Means

\min_{\{c_j\}} \sum_i ||x_i - c_{z_i}||^2

GMM / EM

Q(\theta,\theta^{old}) = E_{Z|X,\theta^{old}}[\log p(X,Z|\theta)]

HMM

P(X_{1:T}, Z_{1:T}) = P(Z_1)\prod_t P(Z_t|Z_{t-1})P(X_t|Z_t)

Kalman Filter

K_k = P^-_k H^T (H P^-_k H^T + R)^{-1}

Wiener Filter

W(\omega)=\frac{S_{xd}(\omega)}{S_{xx}(\omega)}
