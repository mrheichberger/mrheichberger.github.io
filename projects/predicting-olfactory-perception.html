Predicting Olfactory Perception from Molecular Structures

This project investigates whether molecular structure alone can be used to predict human olfactory perception. Molecules are modeled as graphs and learned using graph neural networks to capture structure–perception relationships relevant to electronic nose systems.

Graph Neural Networks Molecular Graphs Machine Learning

Technical Approach

  • Converted molecular representations into graph structures using RDKit
  • Engineered atom- and bond-level features for learning
  • Trained GNN models using PyTorch Geometric
  • Evaluated performance across olfactory perception attributes

Tools & Technologies

Python · PyTorch Geometric · RDKit · NumPy · Matplotlib
