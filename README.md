# mrheichberger.github.io

Personal portfolio and research site.

## Repository Structure

- `index.html` -> main landing page
- `projects/` -> public technical and research project pages
- `resume.pdf` -> public resume artifact
- `assets/` -> public media, visuals, and supporting files (recommended)
- `research/` -> public research writeups and technical notes (recommended)

## Privacy Boundary (Important)

The following are intentionally ignored via `.gitignore` and should remain local/private:

- `private/`
- `planning/`
- `dashboards/`
- `notes/`
- roadmap and planning exports (`*.xlsx`, `*.csv`, etc.)

This keeps personal productivity systems private while keeping public technical work visible on GitHub.
